== Description ==

ShortUrl Tracker allow you to create Short URL directly from your WordPress administration. You can search for product, page, categories etc...
It integrates Google Analytics URL Tags but also custom Param. Careful, this plugin works with a license that's deliver directly from
our website **[https://buildyourweb.fr/plugin-shorturl-tracker](https://buildyourweb.fr/plugin-shorturl-tracker)**.

== Features ==

Three licences are available and each proposed specific features. Note that an upper license will include all features from the lower level License.
You can also upgrade or downgrade licence level.

= Free License: =
- **60** links/month
- Generate **QRcode** for each link
- Google Analytics UTM tag
- Total Link created
- Track **QRcode Scan**
- Track Clicks
- 30 days view
- 30 days Historic Links
- Share on Social Media (Facebook, Twitter, Linkedin, Mail)

= Medium License: =
- **240** links/month
- All Free license features
- Date range **unlimited**
- Track **Device**
- Add custom Param
- Search and Add product variation
- Give **name** to Links
- Attach Link to campaign
- Get **Top Link** (on Global view & by Campaign)
- Create **Campaigns**
- Campaign Dashboard
- Links **Attached to Campaign**

= Premium License: =
- **Unlimited** links/month
- All Free & Medium licenses features
- **Cookie Lifetime** specification
- **Woocommerce** integration
- **Sales** Tracking
- **Global Sales** Tracking
- All **next evolution**


== Installation instructions : ==

1. Download and activate the plugin from the administration of your WordPress site.
2. Go to the plugin configuration page in the "Settings" menu of your WordPress site.
3. Go to ShortUrl Tracker > Settings and fill in the license key. Get a licence **[here](https://buildyourweb.fr/plugin-shorturl-tracker)**
